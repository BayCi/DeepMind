# 项目名称：DeepMind
## 项目介绍：“New Thread 编程赛”项目，以“虚拟助手”与用户进行关于商品信息的对话问 答的方式，帮助其梳理逻辑、明确需求，并通过不同时期的对比和统计，帮助用户更正购物方面上有误的自 我认知。
## 所做工作：项目负责人，搭建框架，UI 层及功能实现代码开发
## 亮点/技术：精美的 UI ，MVC 框架，数组、可变数组与字典的复合数据结构(用于储存问题库与选项库)，商品的购买推荐值算法
## 取得成果：NT赛 团队三等奖
